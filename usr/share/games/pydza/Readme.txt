Pydza is an open-source project of a platform "mario-like" game that will
feature different levels with nice graphics and music (we hope so). Pydza
is written in Python using the pygame library.

========================================================

 System requirement.

You just need a simple computer with
- Python (http://www.python.org/)
- Pygame (http://www.pygame.org/)
It can be run on Windows, Linux or Mac (tested on all of those)

========================================================

 Key bindings

-In Menus

Up/Down arrow:		Select menu item.
Enter:			Perform selected action

-In game

Left/Right arrow:	Move the player in desired direction
Up arrow:		Jump / Wall-jump
Left shift:		Run
Escape:			Pause w/ menu
